Tagex

Tagex is an extensible system for attaching explicit semantics to struct tags. It provides a small execution model built around tags, directives, parameters, and converters.

Core concepts

A Tag defines a processing context. Directives registered on a tag are invoked when struct fields annotated with that tag are processed.
